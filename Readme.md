# 100 Days of Code – Eminent Brands

This repository contains my daily progress, projects, and notes for the  **100 Days of Code Program by Eminent Brands** .

The program is designed to build strong foundations in JavaScript, TypeScript, MERN, Python, Data Science, and Machine Learning.

---

## Program Structure

### Phase 1: Foundations (Days 1–10)

* Git and GitHub fundamentals
* JavaScript basics, closures, scopes
* DOM manipulation
* Mini JavaScript projects

### Phase 2: Advanced JavaScript and DSA (Days 11–25)

* Big-O notation
* Arrays, strings, sets, maps
* Linked lists, stacks, queues
* Trees and graphs
* Sorting and searching algorithms
* Daily DSA problem solving

### Phase 3: TypeScript (Days 26–35)

* TypeScript environment setup
* Types, interfaces, generics
* OOP in TypeScript
* Converting JavaScript projects to TypeScript
* Building simple TypeScript libraries

### Phase 4: Wix Velo Development (Days 36–50)

* Wix ecosystem overview
* Velo frontend and backend
* Data collections and CRUD operations
* Authentication and permissions
* External API integration
* Full Wix Velo application

### Phase 5: MERN Stack Development (Days 51–70)

* React components, props, state
* Hooks and routing
* Express and REST APIs
* MongoDB and Mongoose
* JWT authentication
* Full MERN application
* Deployment and CI/CD basics

### Phase 6: Python and Data Science (Days 71–85)

* Python basics to OOP
* NumPy and Pandas
* Data cleaning and transformations
* Matplotlib visualizations
* Exploratory data analysis
* Automation and scripting tasks

### Phase 7: AI/ML and Neural Networks (Days 86–100)

* Supervised and unsupervised learning
* Regression and classification
* Clustering
* Feature engineering
* Neural networks with TensorFlow/Keras
* Model training, evaluation, and deployment

---

## Daily Workflow

1. Complete the assigned learning tasks
2. Solve 1–2 challenges
3. Push the code to GitHub
4. Post a LinkedIn update
